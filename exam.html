Math Test – 5 Levels (Log, Sets, Functions, Lines)

Level 1 – Easy

Time Left: 60:00

Section A (1×5 = 5)

  1. \(\log_{10}(100)\)
  2. Power set of \(\{0, 1\}\)
  3. Is \(f(x) = x + 1\) a function?
  4. Slope of line \(y = 3x + 2\)
  5. \(A = \{2,4,6\}, B = \{4,6,8\}\). Find \(A \cap B\)

Section B (2×5 = 10)

  1. Solve: \(\log_{10}(x) = 2\)
  2. List all elements in \(A = \{x \in \mathbb{N} : x < 5\}\)
  3. \(f(x) = 2x\). Find \(f(3)\) and \(f(5)\)
  4. Line with slope 2 through point (0, 1)
  5. Is \(R = \{(1,1), (2,2), (3,3)\}\) reflexive?

Section C (3×5 = 15)

  1. Simplify: \(\log_2(8) + \log_2(4)\)
  2. Find \(A \cup B\) and \(A \cap B\), where \(A = \{1,2,3\}, B = \{2,3,4\}\)
  3. Graph \(f(x) = x + 1\) for \(x = -2\) to \(2\)
  4. Slope of line joining (2,3) and (4,7)
  5. Equation of line through (0,0) and (2,3)

Level 2 – Easy-Medium

Time Left: 60:00

Section A

  1. \(\log_{10}(1000)\)
  2. \(A = \{1,2\}, B = \{2,3\}\): Find \(A \cup B\)
  3. Define a function in words.
  4. Find slope of line joining (1,1) and (2,2)
  5. Find domain of \(f(x) = \frac{1}{x}\)

Section B

  1. Simplify: \(\log_{2}(4) + \log_{2}(8)\)
  2. List elements of \(A = \{x \in \mathbb{N} : x \leq 6\}\)
  3. Find \(f(1), f(2), f(3)\) for \(f(x) = x^2\)
  4. Equation of line with slope 1 and passing through (1,2)
  5. Is the relation \(R = \{(a,b) : a = b\}\) symmetric?

Section C

  1. Find number of subsets of \(\{a, b, c\}\)
  2. Check whether \(f(x) = x^2\) is one-one
  3. Find line perpendicular to \(y = -2x + 1\) through (0,0)
  4. Evaluate \(\log_{5}(25)\)
  5. Graph \(f(x) = |x|\) for \(x = -3\) to \(3\)

Level 3 – Medium

Time Left: 60:00

Section A

  1. Find \(\log_{10}(0.01)\)
  2. Is \(f(x) = \frac{1}{x-1}\) defined at \(x=1\)?
  3. Slope of horizontal line
  4. Intersection of \(A = \{1,2\}, B = \{2,3\}\)
  5. Image of 4 under \(f(x) = x^2 + 1\)

Section B

  1. Solve: \(\log_{2}(x) = 3\)
  2. Set of all subsets of \(\{a, b\}\)
  3. Is \(f(x) = x^3\) increasing?
  4. Equation of line through (2, 3) and slope -1
  5. Check if \(R = \{(a,b) : a - b \text{ is even}\}\) is symmetric

Section C

  1. Simplify \(\log\left(\frac{a^2}{b}\right)\)
  2. Determine domain of \(f(x) = \sqrt{x - 2}\)
  3. Find angle between lines with slopes 1 and -1
  4. Line passing through (2, -1) and (4, 3)
  5. Draw graph of \(f(x) = x^2 - 1\)

Level 4 – Medium-Hard

Time Left: 60:00

Section A

  1. Simplify: \(\log_{3}(9)\)
  2. Find cardinality of \(A = \{a,b,c,d\}\)
  3. Slope of vertical line
  4. Check if \(f(x) = \frac{1}{x}\) is odd
  5. Find \(A \setminus B\) where \(A = \{1,2,3\}, B = \{2,3,4\}\)

Section B

  1. Check injectivity of \(f(x) = x^3 + 1\)
  2. Graph \(f(x) = -x\)
  3. Find slope of line through (-2, 4) and (1, 1)
  4. Simplify: \(\log_{10}(10000)\)
  5. Define relation and give example

Section C

  1. Equation of line parallel to \(y = 2x + 5\) through (1, 1)
  2. Check if \(R = \{(a,b) : a^2 = b^2\}\) is symmetric
  3. Simplify: \(\log(ab) - \log(b)\)
  4. Draw \(f(x) = \sqrt{x}\)
  5. Find distance between (0,0) and (3,4)

Level 5 – Hard

Time Left: 60:00

Section A

  1. Evaluate \(\log_{2}(\frac{1}{8})\)
  2. Define equivalence relation
  3. Domain of \(f(x) = \frac{1}{x^2 - 4}\)
  4. Slope between (1,5) and (4,-1)
  5. Find \(f(2)\) for \(f(x) = |x - 3|\)

Section B

  1. Check if \(f(x) = \frac{1}{x}\) is onto in \(\mathbb{R} \setminus \{0\}\)
  2. Prove: \(\log(a^2) = 2\log a\)
  3. Find inverse of \(f(x) = 2x + 5\)
  4. Equation of line through (1,1) with slope \(-\frac{3}{2}\)
  5. Check if relation \(R = \{(a,b) : a + b = 0\}\) is transitive

Section C

  1. Graph \(f(x) = \frac{1}{x}\)
  2. Prove: \(f(x) = x^2\) is not one-one
  3. Find value of \(k\) so that A(1,2), B(3,6), C(k,10) are collinear
  4. Simplify: \(\log(\frac{a^3b^2}{c})\)
  5. Check if \(f(x) = \sqrt{x}\) is defined for all real numbers
